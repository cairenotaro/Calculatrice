# Calcolatrice

Una semplice calcolatrice sviluppata per scopi didattici e di pratica di programmazione.

## Funzionalità

- Operazioni di base: addizione, sottrazione, moltiplicazione e divisione
- Interfaccia semplice e intuitiva
- Codice open source per contribuire e imparare

## Come usare

1. Clona il repository:
   ```
   git clone https://github.com/cairenotaro/Calculatrice.git
   ```
2. Accedi alla cartella del progetto:
   ```
   cd Calculatrice
   ```
3. Segui le istruzioni nel file principale per eseguire la calcolatrice.
